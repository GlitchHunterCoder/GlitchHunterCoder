# Major Projects

### Async Engine

Custom asynchronous execution system designed to run tasks deterministically.

Features:

* generator task support
* queue based scheduling
* safe execution ordering
* lightweight runtime

Status: Release

---

### Perfect Interrupt Safety

Research project investigating how runtime execution can be safely interrupted and resumed.

Concepts explored:

* deterministic execution checkpoints
* execution state preservation
* safe interruption boundaries

Status: Release

---

### Bloxd Game Engine

Experimental scripting engine designed for Bloxd environments.

Goals:

* sandbox safe execution
* high performance runtime
* modding support

Status: Discover

# Runtime Experiments

### Proxy Logging Framework

Deep object graph inspection using layered proxy systems.

Capabilities:

* nested object tracking
* mutation logging
* runtime behaviour tracing
* cyclic graph handling

---

### JavaScript Memory Experiments

Testing how different structures affect observable runtime memory behaviour.

Structures tested:

* primitives
* arrays
* nested objects
* mixed structures

Findings suggest results are heavily influenced by pointer references rather than raw allocation.
